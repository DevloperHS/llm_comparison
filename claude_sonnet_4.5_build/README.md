## Quick Start - The app

Clone the repo :
```
git clone https://github.com/DevloperHS/llm_comparison.git
```

### 1.  Install Dependencies

```bash
cd frontend
npm install
```

### 2. Run 

```bash
npm run dev
```

Access it at localhost!
